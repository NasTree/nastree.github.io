##关于博客
最早接触的博客是上初中，那时候是<a href='https://www.51.com' target='_blank'>51博客</a>，刚刚输入网址试了一下，现在已经变成了游戏网站。同时期的还有QQ空间，前段时间翻了翻那时候写的文章，尴尬癌都犯了，索性全设置成了私密。高中时代，迷恋黑客技术，由于没有电脑只有手机天天在<a href='https://wap.3ghack.com' target='_blank'>黑客契约</a>论坛泡着（已于2012年关闭），修改各种手机软件，刷各种钻，注册大串QQ号寻找连号，满满的成就感。大学时一直有自己搭建博客的想法，由于不懂技术一直没有实施。

###Wordpress
直到后来工作，喜欢上了摄影，又有了搭建个人博客的想法。最早接触的博客系统是<a href='https://cn.wordpress.org' target='_blank'>Wordpress</a>，买空间，买域名，一顿S操作。Wordpress的高度自定义，让我一直在主题上折腾，让强迫症的我重装了无数次博客系统，文章没几篇，折腾的太累了，就放弃了。

###Bitcron
一次偶然，在百度搜索优秀的个人网站，看到了<a href='https://blog.shuiba.co' target='_blank'>水八口</a>的博客，正好在我的审美点上，看到了页脚的博客系统<a href='https://www.bitcron.com' target='_blank'>Bitcron</a>。可惜注册需要邀请码，东求西求还是被我找到了，算是另一翻折腾的开始（虽然有一些自带的简洁的主题，但还是不如意）。各种查阅，也只有<a href='https://blog.shuiba.co' target='_blank'>水八口</a>大神那里有主题制作教程，明白了主题制作的框架，但更细致的问题还是不懂，由于时间成本的原因，干脆在大神那里买了一套主题（尊重版权，买一赠一划得着哈哈～）。

###Farbox2.0
再后来，Bitcron上的一篇公告，意识到博客又要搬迁了，左等右等，没有等到邀请码。然后惊喜的发现<a href='https://farbox.org' target='_blank'>Farbox2.0</a>是开源的，等不急的我还有一个闲置的服务器，索性就自己搭建了起来（虽然后来收到了邀请码，打算折腾够了再搬入官方节点）。由于Farbox2.0是Bitcron的升级版，当时购买的主题也有了用武之地。套上主题，就有了现在的博客。

##关于品牌
在工作的这些年里，一有时间就会去逛一些文艺的小众的店，深知个人品牌对于设计行业的重要性。于是在2017年有了第一个品牌——山月。2019年无意中有了另一个品牌——那是树。

###山月
> 秋高山月白，
> 春至烧痕青。
月圆即团圆，团圆即心安。心安之所，便是归处。正好契合装修设计的本意。设计了logo，注册下域名somoon.cn（已闲置），申请了微信公众号（已停止更新）。

###那是树
> 小屋装点了山，什么来装点小屋呢？那是树！
稚嫩的一句话，引入了这个品牌。为什么说这个品牌是无意中有的呢？因为当时折腾NAS，为了给NAS远程登录，注册了nastree.cn这个域名，谐音“那是树”就有了，而且觉得挺文艺的，就打算用在设计品牌上。同样的设计了logo以及开通了微信公众号（虽然不怎么更新，但是存在）。

注册域名nastree.cn的同时，顺便注册下了nasapp.cn这个域名。考虑到NAS的发展趋势，也许第三方套件需要，也算是一个小投资（哈哈～优惠出手，要不就砸手里了）。
